You have a lot of stuff to buy, and you're tired of haggling. Let's write a function to help, but first, the basics:

Like the functions we were using before, we'll need a name, parameter name and a return value.

	function functionName(parameterName)
	{
		// magic here
		return result;	
	}

The 'result' above can be many things (number, string, etc.), but here we're assuming it's a variable that was set during the magic.
